# 공인 IP vs 사설 IP

---

## 1. 공인 IP (Public IP)

- ISP(인터넷 서비스 제공자)에 의해 할당
- 인터넷 상에서 고유한 주소
- 인터넷에서 직접 접근 가능
- 전 세계적으로 유일
- 예시: 203.0.113.5

---

## 2. 사설 IP (Private IP)

- 로컬 네트워크에서 사용
- 인터넷에서 직접 접근 불가능
- 내부 네트워크 내에서만 유효
- 여러 네트워크에서 중복 사용 가능
- 예시: 192.168.1.10, 10.0.0.1

---

## 3. 사설 IP 주소 범위

- 10.0.0.0 ~ 10.255.255.255 (10.0.0.0/8)
- 172.16.0.0 ~ 172.31.255.255 (172.16.0.0/12)
- 192.168.0.0 ~ 192.168.255.255 (192.168.0.0/16)

---

## 4, 공인 IP vs 사설 IP 비교

| 특징               | 공인 IP                         | 사설 IP                                  |
|--------------------|----------------------------------|-------------------------------------------|
| 용도               | 인터넷 상에서 고유 식별자       | 로컬 네트워크 내 장치 식별               |
| 주소 할당 방법     | ISP에 의해 할당                 | 네트워크 관리자 또는 라우터에 의해 할당 |
| 인터넷 접근 여부   | 가능                             | 불가능                                    |
| 주소 중복 여부     | 중복 불가                        | 여러 네트워크에서 중복 가능              |
| 유효 범위          | 전 세계                         | 내부 네트워크                            |
| 보안 측면          | 노출 위험 있음                  | 내부에서 더 안전                          |
| IP 부족 해결 여부  | 해결 어려움                      | 해결 가능                                 |
