# 네트워크

---

## 1. 네트워크란?

- 컴퓨터 네트워크는 통신 및 데이터 교환을 위해 다양한 장치들이 상호 연결된 시스템
- 특징
  - 프로그램과 데이터를 공유
  - 프린터나 스캐너 같은 주변 장치를 공동으로 사용
  - 인터넷 회선을 여러 대의 컴퓨터가 함께 활용 
  - 데이터를 쉽게 백업할 수 있음

---

## 2. 네트워크의 주요 장단점

### 장점

- 방대한 자료 공유
- 디지털 미디어 접근
- 프로세스 분배를 통한 성능 향상

### 단점

- 보안 문제로 인한 개인정보 유출 위험
- 데이터 변조 가능성

---

## 3. 네트워크의 핵심 구성 요소

- 라우터
  - 서로 다른 네트워크 간에 데이터 패킷을 전달하는 장치
  - 최적의 경로를 결정
- 호스트
  - 네트워크에 연결되어 데이터를 송수신할 수 있는 모든 장치
- 클라이언트
  - 서버에 서비스나 자원을 요청하는 장치 또는 소프트웨어 응용 프로그램
- 서버
  - 네트워크의 다른 장치에 서비스나 자원을 제공하는 컴퓨터 또는 소프트웨어 응용 프로그램
- IP 주소
  - 네트워크에 연결된 각 장치에 할당된 숫자 식별자
  - 장치를 식별하고 데이터를 송수신하는 데 사용
- 스위치
  - LAN 내의 장치를 연결하고 장치 간에 데이터 패킷을 전달하는 장치
  - 네트워크 통신의 효율성을 향상

---

## 4. 네트워크의 회선 구성 방식

- 회선 구성 방식은 컴퓨터와 여러 대의 단말기를 연결하는 방식
    - 포인트 투 포인트 (Point-To-Point) 방식
        - 중앙 컴퓨터와 단말기를 1:1로 연결
        - 언제든지 데이터 전송 가능
    - 멀티 드롭 (Multi-Drop) 방식
        - 멀티 포인트(Multi-Point) 방식이라고도 함
        - 다수의 단말기를 한 개의 통신 회선에 연결
    - 회선 다중 방식
        - 여러 대의 단말기를 다중화 장치를 통해 중앙 컴퓨터와 연결

---

## 5. 네트워크의 데이터 교환 방식

- 회선 교환 방식
    - 통신을 원하는 두 지점을 교환기를 이용하여 물리적으로 접속
    - 예시: 음성 전화망
- 공간 분할 교환 방식
    - 기계식 접점과 전자 교환기의 전자식 접점을 이용하여 교환 수행
    - 예시: 음성 전화용 교환기
- 시분할 교환 방식
    - 전자부품의 고속성과 디지털 교환 기술을 활용
    - 다수의 디지털 신호를 시분할적으로 동작시켜 다중화

---

## 6. 네트워크 통신 방식

- 유니캐스트: 네트워크에서 특정 대상과 1:1 통신하는 방법
- 멀티캐스트: 네트워크에서 특정 대상들과 1:N 통신하는 방법
- 브로드캐스트: 네트워크에서 모든 대상과 통신하는 방법
