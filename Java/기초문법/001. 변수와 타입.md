## 1. 변수

### 변수란

- 변수(Variable)는 값을 저장할 수 있는 메모리의 공간으로, 프로그램 실행 중 값이 바뀔 수 있다.
- 변수는 하나의 값만 저장 가능하며, 하나의 타입만 가질 수 있다.

### 변수의 선언

- 변수를 사용하려면 먼저 타입과 이름을 지정해 선언해야 한다.

```java
int age;      // 정수값 저장
double value; // 실수값 저장
```

- 같은 타입의 변수는 콤마(,)로 함께 선언 가능:
  ```java
  int x, y, z;
  ```
- 변수 이름은 메모리 주소를 대신하는 이름이다.
- 명명 규칙:
    1. 첫 글자는 문자, '$', '_'만 가능 (숫자 시작 불가)
    2. 대소문자 구분
    3. 첫 단어는 소문자, 이후 단어 첫 글자는 대문자 (camelCase)
    4. 예약어 사용 불가
  - 의미 있는 변수명 사용 권장, 한글 변수는 지양.

### 변수의 사용

#### 변수값 저장

- `=` 연산자는 오른쪽 값을 왼쪽 변수에 **저장**한다.

```java
int score = 90;
```

#### 리터럴(Literal)

- 코드에서 직접 입력한 값. 종류에 따라 다음과 같이 구분된다:

| 리터럴 종류 | 예시 | 타입 |
|--------------|------|------|
| 정수 | 10, 0xA, 02 | byte, short, int, long |
| 실수 | 3.14, 5E7 | float, double |
| 문자 | 'A', '\n' | char |
| 문자열 | "Hello" | String |
| 논리 | true, false | boolean |

#### 변수값 읽기

- 변수는 반드시 초기화 후 사용해야 한다.

```java
int value = 30;
int result = value + 10;
```

### 변수의 사용 범위

- 변수는 선언된 중괄호 {} 블록 내에서만 유효하다.

```java
public class Example {
    public static void main(String[] args) {
        int v1 = 15;
        if (v1 > 10) {
            int v2 = v1 - 10;
        }
        int v3 = v1 + v2 + 5; // 오류: v2는 if 블록 밖에서 사용 불가
    }
}
```

## 2. 타입 변환

- 타입 변환은 데이터 타입을 다른 타입으로 바꾸는 것이다. 
- 자동(묵시적) 변환과 강제(명시적) 변환으로 나뉜다.

### 자동 타입 변환 (Promotion)

- 작은 크기 타입 → 큰 크기 타입으로 저장될 때 자동으로 변환됨
- 크기 비교:  
  - `byte(1) < short(2) < int(4) < long(8) < float(4) < double(8)`  
  - float이 int보다 큰 이유는 표현 범위가 더 넓기 때문
- 예시
  ```java
  byte b = 10;
  int i = b;        // 자동 변환
  char c = 'A';
  int n = c;        // 65 저장
  int x = 200;
  double d = x;     // 200.0
  ```
- 예외:
  - `byte → char` 자동 변환 불가 (byte는 음수 허용, char는 0~65535만)

### 강제 타입 변환 (Casting)

- 큰 크기 타입 → 작은 크기 타입으로 변환할 때 사용
- 캐스팅 연산자 `(타입)` 사용

  ```java
  int i = 103029770;
  byte b = (byte) i; // 강제 변환
  ```
- 변환 시 상위 비트가 버려질 수 있어 데이터 손실 가능
- 예시
  ```java
  double d = 3.14;
  int i = (int) d; // 3만 저장
  ```
- 변환 전 값이 대상 타입의 범위를 벗어나면 변환하지 말아야 함

  ```java
  int i = 128;
  if (i < Byte.MIN_VALUE || i > Byte.MAX_VALUE)
      System.out.println("변환 불가");
  else
      byte b = (byte) i;
  ```
- 정밀도 손실 주의:
  ```java
  int num1 = 123456780;
  float num2 = num1;   // 근사치 저장 → 손실 발생
  num2 = (int) num2;   // 원래 값과 다름
  ```
- 해결: `double` 사용

### 연산식에서의 자동 타입 변환

- 서로 다른 타입의 피연산자 → 큰 타입으로 자동 변환 후 연산
  ```java
  int i = 10;
  double d = 5.5;
  double result = i + d; // int → double 변환
  ```
- 정수 연산 기본은 `int`.  
  `byte`, `short`, `char`도 연산 시 자동으로 `int`로 변환됨.
  ```java
  byte a = 10, b = 20;
  int c = a + b; // 결과는 int
  ```
- char 연산 결과는 int로 나오므로 다시 문자로 쓰려면 캐스팅 필요
  ```java
  char ch = 'A';
  char next = (char)(ch + 1); // 'B'
  ```
- long 포함 시 결과는 long, double 포함 시 결과는 double.
