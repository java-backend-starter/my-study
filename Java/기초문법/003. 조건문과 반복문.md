# 기초문법(조건문과 반복문)

---


## 1. 조건문

### if문

- 조건식이 true일 때 블록 실행, false면 실행하지 않음.
- 중괄호 { }로 여러 실행문을 묶을 수 있음.
- 한 줄 실행문이면 생략 가능하지만, 생략 시 의도치 않은 실행 오류 발생 가능.
```java
if(score >= 90) {
    System.out.println("점수가 90 이상입니다.");
}
```

### if-else문

- 조건식이 true이면 if 블록 실행, false이면 else 블록 실행.
- 두 가지 경우만 처리할 때 간결하게 사용 가능.
```java
if(score >= 90) {
    System.out.println("점수가 90 이상입니다.");
} else {
    System.out.println("점수가 90 미만입니다.");
}
```

### if-else if-else문

- 여러 조건을 순차적으로 판단 가능.
- 첫 번째 true 조건 블록만 실행 후 전체 if문 종료.
- 마지막 else는 모든 조건이 false일 때 실행.
```java
if(score >= 90) {
    System.out.println("A");
} else if(score >= 80) {
    System.out.println("B");
} else {
    System.out.println("C");
}
```

### 중첩 if문

- if문 안에 또 다른 if문 사용 가능.
- 조건을 세부적으로 나눠서 처리할 때 유용.
```java
if(score >= 90) {
    if(score >= 95) grade = "A+";
    else grade = "A";
} else {
    if(score >= 85) grade = "B+";
    else grade = "B";
}
```
- 중첩 단계가 깊어질수록 실행 흐름 이해 필요.

### switch문

- 변수 값에 따라 실행 블록 선택.
- if-else보다 경우의 수가 많을 때 코드 간결.
- 기본 구조:
```java
switch(variable) {
    case 값1:
        실행문;
        break;
    case 값2:
        실행문;
        break;
    default:
        실행문;
}
```
- break 없으면 다음 case도 연속 실행.
- 사용 가능 타입:
    - 자바 6 이하: byte, char, short, int
    - 자바 7 이상: String도 가능
- char, String 등 다양한 타입에서 동일 값 처리 가능.
```java
switch(grade) {
    case 'A': case 'a':
        System.out.println("우수 회원");
        break;
    }
```

###  난수 활용 if/switch

- Math.random() 사용: `0.0 <= Math.random() < 1.0`
- 정수 범위 난수 생성: `int num = (int)(Math.random() * n) + start;`
- 주사위 예제: `int num = (int)(Math.random()*6) + 1;`

## 2. 반복문

- 반복문: 같은 작업을 여러 번 수행할 때 사용.
- 종류: for, while, do-while.
- 선택 기준
    - for: 반복 횟수가 정해져 있을 때.
    - while: 조건에 따라 반복할 때.
    - do-while: 최소 한 번은 실행 후 조건 검사.

### for문

- 형식: `for(초기화식; 조건식; 증감식) { 실행문 }`
- 특징
    - 초기화 → 조건 검사 → 실행 → 증감 → 조건 검사 반복.
    - 반복 횟수가 명확할 때 사용.
    - 초기화식 생략 가능.
    - 변수 범위: 초기화식에서 선언한 변수는 블록 내에서만 사용 가능.
    - 중첩 가능: 외부 루프 한 번 실행 시 내부 루프 반복.
    - 주의: float/double 타입 카운터 사용 시 오차 발생 가능.
- 예제
```java
for(int i = 1; i <= 10; i++) {
    System.out.println(i);
}
```

### while문

- 형식: `while(조건식) { 실행문 }`
- 특징
    - 조건이 true일 때 반복.
    - 조건 검사 후 실행.
    - 무한 루프 가능 (while(true)) → 탈출 조건 필요.
    - 반복 중 변수는 반드시 루프 외부에서 선언.
- 예제
```java
int i = 1, sum = 0;
while(i <= 100) {
    sum += i;
    i++;
}
```

### do-while문

- 형식 `do {  // 실행문  } while(조건식);`
- 특징
    - 최소 한 번은 실행 후 조건 검사.
    - 조건 검사 결과가 true면 반복.
    - 키보드 입력 등 먼저 실행 후 조건 검사가 필요한 경우 적합.

### break문

- 반복문이나 switch문을 즉시 종료.
- 중첩 반복문: 가장 가까운 반복문만 종료.
- 라벨(label) 사용 시 바깥 반복문까지 종료 가능.
- 예제
```java
while(true) {
    int num = (int)(Math.random()*6)+1;
    if(num == 6) break;
}
```

### continue문

- 반복문 종료 없이 다음 반복으로 이동.
- 조건 만족 시 이후 문장은 실행되지 않고 다음 반복으로 넘어감.
- 예제
```java
for(int i = 1; i <= 10; i++) {
    if(i % 2 != 0) continue; // 홀수 스킵
    System.out.println(i);   // 짝수만 출력
}
```
