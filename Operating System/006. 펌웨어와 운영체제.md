# 펌웨어와 운영체제

---

## 1. 펌웨어(Firmware)

- 하드웨어를 제어하고 소프트웨어와의 중간 역할을 수행하는 프로그램
- 보통 비휘발성 메모리(예: 플래시 메모리)에 저장되어 있음

---

## 2. 펌웨어의 주요 특징

- 하드웨어 제어: 장치의 기본 동작을 담당 (예: 버튼 동작, 입력 처리 등)
- 업데이트 가능: 보안 및 기능 향상을 위해 주기적으로 업데이트
- 내장형 저장: 장치 내부의 비휘발성 메모리에 저장되어 전원 꺼져도 유지됨

---

## 3. 펌웨어 예시

### 스마트폰의 펌웨어

- 스마트폰에는 다양한 펌웨어가 내장되어 있음
- 예시
  - 카메라 모듈을 제어하는 펌웨어는 카메라의 촬영, 줌 기능 등을 관리
  - 배터리 충전 펌웨어는 배터리 상태를 모니터링하고 과충전을 방지

### 네트워크 라우터 펌웨어

- 라우터에는 인터넷 연결 설정, 트래픽 관리, 보안 기능 등을 제어하는 펌웨어가 존재
- 펌웨어 업데이트를 통해 새로운 네트워크 프로토콜을 지원하거나 보안 취약점을 해결 가능

### 프린터 펌웨어

- 프린터 펌웨어는 인쇄 작업을 제어하며, 종이 크기나 해상도 조절 같은 설정을 관리 
- 업데이트를 통해 인쇄 품질을 향상시키거나 새로운 인쇄 기능을 추가 가능

### 자동차 ECU (전자제어장치)

- 자동차의 전자제어장치는 여러 개의 ECU가 있으며, 엔진, 브레이크, 에어백 등 다양한 시스템을 제어 
- ECU 펌웨어 업데이트를 통해 차량 성능을 개선하거나 엔진 소프트웨어 오류를 수정 가능

---

## 4. 운영체제와 비교

| 항목 | 펌웨어 | 운영 체제 (OS) |
|------|--------|----------------|
| 저장 위치 | 보통 ROM에 저장됨 | RAM과 저장장치(SSD/HDD)에 존재 |
| 규모 | 특정 하드웨어의 최소 동작을 위한 작은 프로그램 | 다양한 하드웨어 및 소프트웨어를 관리하는 대규모 프로그램 |
| 업데이트 빈도 | 거의 업데이트되지 않으며, 방법도 복잡하고 특수함 | 자주 업데이트되며, 수정이 비교적 간단함 |
| 적용 기기 | 거의 모든 전자기기 (신호등, 계산기, 서명기 등) | 컴퓨터, 스마트폰 등 복잡한 작업이 필요한 전자기기 |
