# 에자일 vs 폭포수

---

## 1. 폭포수 vs 애자일 비교

| 구분             | 폭포수 모델                                                                 | 애자일 모델                                         |
|------------------|----------------------------------------------------------------------------|-----------------------------------------------------|
| 기본 가정        | 시스템을 전체적으로 정의하고 예측 가능한 세부 계획에 따라 개발               | 지속적인 설계 개선과 테스트, 빠른 피드백, 변경을 통해 개발 |
| 관리 유형        | 명령과 통제                                                                 | 리더십과 협력                                       |
| 지식 관리        | 명시적 지식                                                                 | 암묵적 지식                                         |
| 의사소통         | 공식적                                                                      | 비공식적                                            |
| 조직 구조        | 관료적이고 대규모 조직                                                       | 유연하고 참여적인 중소 규모 조직                     |
| 품질 관리        | 계획 수립이 어렵고 엄격한 통제, 테스트가 어렵고 오래 걸림                     | 지속적인 통제, 요구사항, 설계, 해결, 테스트 포함     |
| 사용자 요구사항  | 수행 전에 상세하게 정의                                                       | 상호작용에 따라 정의                                 |
| 재시작 비용      | 높음                                                                        | 낮음                                                |
| 개발 방향        | 고정됨                                                                      | 쉽게 변경 가능                                      |
| 테스트 시점      | 개발이 완료된 후                                                             | 각 반복 주기마다 수행                                |
| 고객 참여도      | 낮음                                                                        | 높음                                                |
| 구조             | 현재와 미래의 요구사항을 설계                                                 | 현재의 요구사항에 집중하여 설계                      |
| 주 목적          | 높은 안정성                                                                  | 빠른 가치 생성                                      |
