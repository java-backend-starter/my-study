# 정규화와 역정규화

---

### 1. 정규화와 역정규화 비교

| 항목        | 정규화 (Normalization)                      | 역정규화 (Denormalization)                  |
|-------------|----------------------------------------------|----------------------------------------------|
| 목적        | 중복 제거, 무결성 유지                       | 성능 향상, 읽기 속도 개선                   |
| 방식        | 테이블을 분리하고 관계를 명확히 함           | 테이블 통합, 중복 데이터 허용               |
| 장점        | 데이터 일관성 유지, 정합성 보장              | 빠른 조회 성능, 조인 연산 감소              |
| 단점        | 조인 연산 증가로 성능 저하 가능              | 데이터 중복, 무결성 유지 어려움             |
| 사용 시기   | 무결성과 정합성이 중요한 시스템              | 성능이 중요한 시스템, 조회 중심 환경       |
