# 관계형 데이터베이스와 비관계형 데이터베이스

---

## 1. 관계형 DB(RDBMS)

- 사전 정의된 관계를 기반으로 데이터를 테이블(행·열) 형태로 저장
- 기본키와 외래키를 사용하여 테이블 간 연결 가능

---

## 2. 비관계형 DB(NoSQL)

- 특정 목적에 맞춰 유연한 스키마를 갖춘 데이터베이스
- 대용량 데이터 처리, 짧은 지연 시간, 유연한 데이터 모델에 최적화

---

## 3. 차이점

- 저장 방식: 관계형 DB는 테이블 형식, 비관계형 DB는 다양한 데이터 모델 사용
- 확장성: 관계형 DB는 수직 확장, 비관계형 DB는 수평 확장에 강점
- 데이터 일관성: 관계형 DB는 강한 일관성 보장, 비관계형 DB는 성능을 위해 일부 제한 완화
